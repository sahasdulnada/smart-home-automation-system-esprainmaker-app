

This project delivers a fully connected smart home solution that allows users to control four AC appliances through the ESP RainMaker app, Alexa, and Google Assistant. It also supports offline manual control via physical switches, ensuring continuous operation even without internet connectivity.

The system integrates seamlessly with popular IoT platforms like IoT Cloud and Firebase, enabling enhanced data logging and remote monitoring. An onboard OLED display and status LEDs provide real-time visual feedback for each appliance, improving usability and system transparency.

Key Features:

Internet-based control via ESP RainMaker app

Voice control using Amazon Alexa & Google Assistant

Offline functionality with manual switches

Integration with IoT Cloud & Firebase

Real-time status monitoring with OLED display & indicator LEDs

Reliable control of four independent AC appliances
